## AniVault
anime library for exploring latest shows, genres, characters, studios, and more

### Features
- responsive app styled with tailwind includes both light and dark themes
- integrates jikan-api for database
- recommendations for finding similar shows 
- collections - users can save shows into custom lists
- Rate and write a review

### Pages
 - Home: landing page with a recent shows cards-slider
 - Auth: login/signup page
 - Show: a page dedicated for single show view includes all info and possible interaction
 - Character: character page shows background and other character info and featured shows
 - Profile: user's page includes their profile-data, their recent history, their shows-status (watched, to be watched, reviews, etc) and their collections
